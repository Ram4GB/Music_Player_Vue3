# Mucsic App Using Vue 3 bản official

Sử dụng ngôn ngữ Vue 3 đơn thuần để viết.

## Chức năng

-   Loop nhạc
-   Chọn nhạc
-   Chọn bài trước đó / sau đó
-   Dừng nhạc
-   Zoom to/ Zoom bé
-   Chỉnh sửa âm thanh
-   Animation lúc bấm play nhạc
